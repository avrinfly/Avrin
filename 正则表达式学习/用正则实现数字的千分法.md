代码实现
```
let num = 2342342.23
let result = num.toFixed(2).replace(/(\d)(?=(\d{3})+\.)/g, '$1,')
```
> 解析  

\d ： 查看数字

?= ： 正向肯定预查，且该预查不消耗字符，也就是说在一个匹配发生后，在最后一次匹配之后立即开始下一次匹配的搜索，而不是从包含预查的字符之后开始。

*通俗的说：匹配完，从结束位置继续开始匹配*

\d{3} 匹配三位数字

+\. 转义后的小数点(.)，区分小数点前和后， + 代表匹配一次或多次
